Se repérer sur le site

Le site est organisé autour d’un menu principal situé en haut de la page. L’onglet Accueil présente une vue générale avec un tableau de bord. L’onglet Données regroupe les pages d’analyse comme les prescriptions, les effectifs, les honoraires, la comparaison et la carte.

La page API Ameli sert de point d’entrée pour rechercher rapidement des données à partir de filtres simples.

Comment utiliser les filtres ?

1

Choisissez une profession de santé dans la liste déroulante.

2

Sélectionnez une région : cela met automatiquement à jour la liste des départements.

3

Choisissez le département et l’année que vous voulez analyser.

4

Cliquez sur Afficher ou Analyser pour obtenir les résultats.

Comment lire les résultats ?

Les résultats sont généralement présentés sous deux formes : un graphique pour visualiser rapidement les tendances, et un tableau pour consulter les données brutes plus précisément.

À retenir

Si aucune donnée ne s’affiche, cela ne veut pas forcément dire qu’il y a une erreur. Il est possible que l’API Ameli ne possède pas de résultat pour la combinaison choisie : profession, département et année.

Conseils d’utilisation

  • Commencez par une profession courante, comme les médecins généralistes ou les infirmiers.
  • Essayez d’abord une grande zone, comme Paris ou un département très peuplé.
  • Changez d’année si aucune donnée n’est trouvée.
  • Utilisez la carte pour obtenir une vue globale par département.
  • Utilisez la comparaison pour observer des écarts entre deux professions ou deux années.
